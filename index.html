Home Assistant

Home Assistant is a versatile platform for automating and controlling smart home devices. It allows you to control almost any smart device from a centralized dashboard and supports local control for increased reliability and security. You can also create custom dashboards for personalized control. With Home Assistant, you can unify and streamline all your smart home needs in one powerful system.

Frigate

Frigate NVR is an advanced video surveillance system designed for smart home security. It uses AI-powered object detection to monitor cameras, allowing you to detect people, vehicles, and other objects with high accuracy. While running efficiently on local hardware, Frigate offers a privacy-focused, low-latency solution for monitoring and recording, making it an ideal choice for smart and secure home surveillance.

Double Take

Double Take is a facial recognition tool that integrates with Frigate NVR and uses systems like CompreFace to identify faces from your camera feeds. It can trigger automations in Home Assistant, such as unlocking doors when a recognized face is detected. Double Take processes images locally for privacy and security, providing a seamless way to enhance smart home automation through reliable facial recognition.

HTML

HTML was the easiest to learn, in my opinion, but from my experience, you can't do much with it alone—so where's the fun? Obviously used to structure a webpage, you'll usually get the gist of HTML after 1-2 projects. HTML gets more interesting once you introduce it to PHP, JavaScript, and CSS, but it still provides the foundation for a webpage, so it's certainly necessary to be familiar enough with it to read and comprehend.

PHP

PHP was introduced as a helpful tool for connecting your site to a database. From using PHP to create data tables to using it as a way to enable login functionality, it certainly pulls its weight in some of my projects. Its only downside is having to write SQL to query the database, because who isn’t slightly intimidated by SQL, especially with a full database?

JavaScript

JavaScript is your best friend when you want to add a little bit more functionality. Whether that be by using it to send input data to your database through PHP, or just using it to make things on your webpage more dynamic and fun. For example, the spinning hamburger menu that's available on this site's mobile version. Try it out!

YAML

IT Helpdesk

During my first year at Ole Miss, I worked at the IT Helpdesk for the University. This is where students and faculty would go for their tech support.
